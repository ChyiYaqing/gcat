# gcat

Use grpc, grpc-gateway, openapi to develop backend template projects

### gRPC

> gRPC Stands for gRPC Remote Procedure Call, gRPC is a modern, open source remote procedure call(RPC) framework that can run anywhere. It enables client and server applications to communicate transparently, and makes it easier to build connected systems.

### gRPC-Gateway

> gRPC-Gateway helps you to ptovide your APIs in both gRPC and RESTful style at the same time.
